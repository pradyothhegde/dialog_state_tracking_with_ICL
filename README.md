# dialog_state_tracking

Following folders contain these things:

- new_scripts
    - Contains sequence of scripts to run for the data preparation, inference and evaluation of dialog state tracking. (as of now MW24 dataset.)

- experiments
    - contains different setup logs in respective folders
- results
    - has same structure as the experiments folder. Single output files are placed in the respective folders of the experiment
    - with respect to the folder / setting there are CSV files containing the results.
